# Statement of the Project

## 1. Problem Statement
Students often forget important tasks such as assignments, homework, exam dates, and project deadlines. This leads to poor time management and stress. A simple task reminder system is needed to help students organize, store, and track their daily academic tasks.

## 2. Scope of the Project
This project focuses on basic task management for students. It includes adding tasks, viewing tasks, editing tasks, deleting tasks, and marking tasks as completed.  
It does NOT include advanced features like push notifications, cloud storage, or multi-user authentication (these can be added in the future).

## 3. Target Users
- School students  
- College students  
- Anyone who needs to track personal study tasks and deadlines  

## 4. Objectives
- To build a simple and user-friendly task reminder system  
- To help students manage their deadlines efficiently  
- To allow adding, editing, viewing, and deleting tasks  
- To ensure tasks are stored safely using file storage  
- To provide a structured workflow for daily task tracking  

## 5. High-Level Features
- Add a new task with title, description, date, and time  
- View all tasks  
- View today's tasks  
- Edit or update task details  
- Delete a task  
- Mark a task as completed
