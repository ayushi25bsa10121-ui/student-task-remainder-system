A simple system to help students manage their daily tasks,deadlines and upcoming academic activities. 

⭐ Overview

⁘ This project allows students to add tasks, view pending tasks, update task status, and manage deadlines in an organized way. It helps students avoid forgetting important academic responsibilities.

⭐ Features

1.Add a new task with title, description, date, and time

2.View all tasks

3.View today's tasks

4.Edit or update task

5.Delete task

6.Mark tasks as completed

⭐ Technologies Used

1.Python

2.JSON / File storage

3.Git & GitHub

⭐ How to Run

1.Install Python

2.Download or clone the repository

3.Run the command: python main.py

4.Follow the on-screen instructions

⭐ Project Requirements (Short Version)

# Problem Statement
1.students often forget tasks and deadlines. This system helps them store and track tasks.

# Objectives
1.Provide easy task management

2.Help students organize tasks

3.Display upcoming deadlines

# Functional Requirements
1.Add, view, update, delete tasks

2.Store tasks in file

3.Show today’s and upcoming tasks

# Non-Functional Requirements

2.Fast performance

3.Proper error handling

4.Reliable storage
